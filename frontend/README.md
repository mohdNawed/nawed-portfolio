# Frontend

The production Vite frontend currently lives in `../src` so Vercel can keep building the existing project safely.

Feature folders now available in the app:

- `../src/components` reusable UI components
- `../src/pages` route-level pages
- `../src/admin` admin screens
- `../src/api` frontend API helper

Use this folder when separating the frontend into a standalone package later.
